# Project: Aplikasi Todo List PHP Native

## Deskripsi Singkat
Aplikasi manajemen tugas sederhana untuk menambah dan menghapus daftar tugas menggunakan PHP Native dan basis data MySQL/MariaDB.

## Lingkungan Pengembangan
- **XAMPP** (Apache & MySQL)
- **Editor**: Visual Studio Code
- **Bahasa**: PHP Native, HTML, CSS

## Struktur Folder
- `config.php`: Pengaturan koneksi ke database.
- `index.php`: Halaman utama dan tampilan daftar tugas.
- `proses.php`: Logika untuk memproses tambah dan hapus data.

## Cara Instalasi
1. Pindahkan folder proyek ke folder `htdocs` di XAMPP.
2. Buat database bernama `db_todo` di phpMyAdmin.
3. Buat tabel `tasks` dengan kolom `id` dan `task_name`.
4. Buka browser dan akses `localhost/tugas_todo`.
